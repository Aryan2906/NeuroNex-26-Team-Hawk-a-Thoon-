


# NeuroSpike: Early Cognitive Decline Detection using Spiking Neural Networks
## Overview

NeuroSpike is an AI-powered healthcare application designed to assist in the early detection of cognitive decline using MRI brain scans and clinical reports.

The system combines the feature extraction capabilities of Convolutional Neural Networks (CNNs) with the energy-efficient processing of Spiking Neural Networks (SNNs) to perform medical image classification while simulating biologically inspired neural behavior.

In addition to MRI analysis, the platform can generate AI-assisted clinical interpretations from uploaded medical reports, providing a unified cognitive assessment experience.

## Problem Statement

Early diagnosis of neurodegenerative conditions such as Alzheimer's Disease is critical for timely intervention and treatment planning.

Traditional diagnostic workflows often require:

Manual MRI interpretation
Extensive neurological testing
Expert clinical assessment

NeuroSpike aims to support clinicians and patients by providing:

Automated MRI classification
AI-generated report summaries
Fast and accessible cognitive risk assessment

## Installation
Clone Repository
git clone https://github.com/yourusername/neurospike.git

cd neurospike
Create Virtual Environment
python -m venv venv
Activate Environment
Windows
venv\Scripts\activate
Linux / macOS
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Environment Variables

Create a .env file in the project root.

OPENAI_API_KEY=your_api_key_here
Running the Application
Launch Streamlit
streamlit run frontend/app.py

The application will start locally and open in your browser.
